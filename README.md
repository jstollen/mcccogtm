# MC CCO GTM Brand Kit

Salesforce Marketing Cloud CCO GTM brand kit for SE-facing release trackers, enablement materials, and GTM collateral.

## Brand Colors

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#032D60` | Headers, headings, primary UI elements |
| Accent | `#0176D3` | Links, buttons, interactive elements |
| SF Blue | `#00A1E0` | Salesforce brand blue (logo, highlights) |
| SF Cloud | `#1B96FF` | Cloud blue (data viz, stats) |
| Background | `#FFFFFF` | Page background |
| Surface | `#F3F3F3` | Cards, panels, secondary backgrounds |
| Foreground | `#181818` | Body text |
| Muted | `#706E6B` | Secondary text, captions |
| Border | `#DDDBDA` | Dividers, borders |
| Success | `#2E844A` | On-Track, Completed statuses |
| Warning | `#FE9339` | Watch status |
| Error | `#EA001E` | At Risk, Canceled statuses |

## Typography

- **Primary**: Inter (Google Fonts)
- **Fallback**: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Helvetica Neue, Arial, sans-serif
- **Headings**: 600-700 weight, -0.02em letter-spacing
- **Body**: 400-500 weight, 1.6 line-height

## Logo

Salesforce wordmark logo. Used in headers on dark navy (#032D60) backgrounds.

## Templates

- `email-template.html` — Email template with `{{content}}` placeholder
- `report-template.html` — Report template with Chart.js support
- `website-template.html` — Website/page template
